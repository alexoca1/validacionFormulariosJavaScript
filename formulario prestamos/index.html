Solicitud de préstamos

Situación Laboral

Haz un formulario ( 1pto ) de Solicitud de préstamos donde se introduzca el nombre, el importe a solicitar, la edad y una lista desplegable “Situación laboral” con la opciones: Elija una, en activo, desempleado, jubilado.

Antes de enviar los datos el formulario debe validar que se ha introducido un nombre, que se ha introducido un importe mayor a 10.000 euros ( 2ptos ) , que la edad es menor de 65 años ( 2ptos ) y que se ha elegido un situación laboral.(2ptos )

( 2 ptos )Los mensajes de error no los muestra en una ventana emergente sino que los pone como contenido de un h3 que se coloca justo a continuación del formulario.

(1 pto) Además si un input da error el foco se coloca en ese input
